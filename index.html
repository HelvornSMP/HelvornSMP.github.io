Frequently Asked Questions

Everything you need to know about joining and playing

Copy the server IP address above and add it to your Minecraft server list. If the server appears offline, try joining twice as it may need to wake from sleep mode.
We support both premium and cracked clients. You can join with or without purchasing Minecraft.
Rules vary by game mode. Anarchy has 1 rule, no hacking. Survival has basic anti-griefing protections. Creative mode is currently under disscussion. Join our Discord for detailed rule information.
Contact an admin through Discord or use in-game commands if you have the required permissions. Each game mode runs on its own dedicated world.
We're currently developing the Lifesteal mode. Join our Discord server to stay updated on the release date and participate in beta testing.
